##代码区块

直接缩进4个空格或一个制表符即可：

function Person(name,job){
    this.name = name;
    this.job = job;
    this._age = 23;
    this.edition = 1;
}
var nikolausliu = new Person('liuyu','Front-End');
Object.defineProperty(nikolausliu,'age',{
    get : function(){
        return this._age;
    },
    set : function(newValue){
        if((newValue + 1994) > 2017){
            this._age = newValue;
            this.edition += newValue + 1994 - 2017;        
        }
    }
});

当然，直接缩进4个空格的格式是markdown原生支持的代码区块写法，若果需要用到代码高亮和圆角、代码行数显示，则需要用到主题定义的写法。我现在用的是next主题,写法是把代码区块放在头尾的连续三个反引号之间
